# 轻待办 (Light To Do)

一款专为 macOS 打造的极致轻量化待办事项应用。它抛弃了传统应用繁杂的界面，通过原生液态玻璃设计语言与智能化交互，让任务管理变得纯净而优雅。部分功能演示视频可访问：https://ivean.com/lighttodo/

## 核心特性

### 边缘吸附交互
这是软件的交互核心。当你将窗口拖动到屏幕的左侧或右侧边缘时，窗口会自动吸附并收缩。
- 悬停滑出：将鼠标移动到屏幕边缘的细长条上，窗口会顺滑滑出。
- 智能收回：如果滑出后未进行操作，鼠标移开即收回；如果有操作，则在点击窗口外部后收回。
- 空间无感：在不使用时完全不占用屏幕空间，真正做到随叫随到。

### 边缘光晕提醒
相比传统的弱弹窗通知，本应用通过视觉流光提供更具质感的提醒体验。
- 呼吸感光晕：当待办事项到期时，屏幕边缘会产生具有呼吸感的彩色流光。
- 多色可选：支持橙、蓝、绿、红、紫等多种色彩配置。
- 沉浸式感知：在不打断当前工作流的前提下，通过环境光感提醒任务状态。

## 设计与交互

### 原生液态玻璃效果
界面全量采用 macOS 现代设计语言，具备极致的半透明模糊与动态光影效果。无论是在浅色还是深色模式下，都能与系统桌面完美融合。

### 自定义时间交互
- 智能识别：在输入待办内容时，系统会自动捕捉其中的日期信息并提供一键设置建议。
- 循环快捷键：时间设置面板提供循环累加按钮。点击可快速设定“1小时后”，持续点击可循环切换至“6小时后”。
- 快速槽位：内置“今晚”、“明天”、“下周一”等基于当前时间的智能快捷项。

### 纯净无边框体验
移除了所有装饰性的标题栏与按钮。整个窗口就像悬浮在桌面上的半透明玻璃片，专注于内容本身。

## 快速开始

### 环境要求
- macOS 13.0 (Ventura) 或更高版本

### 编译与运行
项目提供了自动化的打包脚本，可一键生成原生应用程序包。

```bash
chmod +x build_app.sh
./build_app.sh
```

执行后，脚本将自动完成 React 前端构建与 Swift 编译，并启动生成的 Light To Do.app。

## 数据存储
应用程序数据保存于以下位置，方便用户进行个人备份：
`~/Library/Application Support/LightToDo/notes.json`
